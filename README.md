# Front-end

### 项目背景介绍

#### 我们的项目是一个基于jekyll的个人博客

#### test界面留作日后使用
#### blog界面是静态博客网页。同时我们实现了一个用户管理侧边栏。

### 安装和运行方式
#### 查看gem源
<code>gem sources -l</code>
#### 更新gem
<code>gem update --system</code>
#### 安装jekyll
<code>gem install jekyll</code>
#### 激活bundle
<code>bundle</code>
#### 启动jekyll
<code>jekyll serve</code>
